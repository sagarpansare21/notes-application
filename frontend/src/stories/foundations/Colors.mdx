# Color System

Our color palette is carefully calibrated using modern HSL and OKLCH color spaces, drawing design inspiration from Notion's clean structures and Linear's dark, premium interfaces.

---

## Palette Overview

Each color is mapped to CSS variables and Tailwind classes. We support a reactive Light/Dark theme mapping.
